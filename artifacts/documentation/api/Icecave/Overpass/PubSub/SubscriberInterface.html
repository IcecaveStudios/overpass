consume( callable $callback)

Consume messages from subscriptions.

When a message is received the callback is invoked with two parameters, the first is the topic to which the message was published, the second is the message payload.

The callback must return true in order to keep consuming messages, or false to end consumption.

Parameters

callable $callback The callback to invoke when a message is received.
